# Collaborative Documents

Instead of [insert terrible corporate document collaboration tool here, simply use a Git repo with some md (or [your preferred markup](more.org)) files.
Give your collaborators permission to push directly to main, and use the GitHub web editor.

If the GitHub web editor isn't good enough, you can clone it and edit the files locally in the best text editor (you know which one).

# `Math(s|ematics)?`

Just $\frac 1 2$. No drama. No jumping through menus to find the "insert equation" button, which will *in the best case* give you a plain text input field to type some LaTeX (or maybe AsciiMath if they're cool) anyway.
